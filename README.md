# CppQtNotepad

采用Qt编写的轻量级的Notepad.

## 条件

* VS2022 & Qt6.5.1

## 开始

* 克隆仓库

```
git clone https://github.com/mowangshuying/FluentUI.git
```

* Build

  ```
  1. 用Qt Creator打开CMakeLists.txt
  2. 编译
  ```
